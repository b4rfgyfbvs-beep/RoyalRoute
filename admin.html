To OPEN the site:
1. Go to Vercel → Settings → Environment Variables
2. Set SITE_OPEN = true
3. Set SPOTS_TAKEN = 0
4. Save and redeploy

To CLOSE the site:
1. Go to Vercel → Settings → Environment Variables
2. Set SITE_OPEN = false
3. Set SPOTS_TAKEN = 5
4. Save and redeploy
